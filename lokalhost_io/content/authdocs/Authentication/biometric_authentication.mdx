### 3. Blocks

Blocks are a set of components to make a complete blocks, they contain multiples files. To install them it is mandatory to use the `cli`. For example it you'd like to install `ai-card-generation` block:

<Tabs items={["npx", "bun", "pnpm"]}>
 
```bash tab="npx"
npx shadcn@latest add https://shadifyui/r/ai-card-generation.json
```

```bash tab="bun"
bunx shadcn@latest add https://shadifyui/r/ai-card-generation.json
```
 
```bash tab="pnpm"
pnpm dlx shadcn@latest add https://shadifyui/r/ai-card-generation.json
```

</Tabs>

Then, add to your page and it works!

### 5. Monorepo

For monorepo `shadcn/ui` CLI contain the options `-c` to the path to your workspace for exemple:

<Tabs items={["npx", "bun", "pnpm"]}>
 
```bash tab="npx"
npx shadcn@latest add https://shadifyui/r/card-05.json -c ./apps/www
```
```bash tab="bun"
bunx shadcn@latest add https://shadifyui/r/card-05.json -c ./apps/www
```
 

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://shadifyui/r/card-05.json -c ./apps/www
```

</Tabs>
